# –ö–∞–∫–≤–æ –µ Dictionary –≤ Python?

–í Python —Ä–µ—á–Ω–∏–∫—ä—Ç (dictionary, —Ç–∏–ø dict) –µ –≤–≥—Ä–∞–¥–µ–Ω–∞ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –æ—Ç –¥–∞–Ω–Ω–∏, –∫–æ—è—Ç–æ —Å—ä—Ö—Ä–∞–Ω—è–≤–∞ –¥–≤–æ–π–∫–∏ –æ—Ç –∫–ª—é—á –∏ —Å—Ç–æ–π–Ω–æ—Å—Ç
‚Äî –Ω–µ—â–æ –∫–∞—Ç–æ –º–∏–Ω–∏ –±–∞–∑–∞ –¥–∞–Ω–Ω–∏ –≤ –ø–∞–º–µ—Ç—Ç–∞. 
–†–µ—á–Ω–∏–∫—ä—Ç –µ –∫–∞—Ç–æ –µ—Ç–∏–∫–µ—Ç–∏—Ä–∞–Ω –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä ‚Äî –≤–º–µ—Å—Ç–æ –¥–∞ —Ç—ä—Ä—Å–∏—à –ø–æ –Ω–æ–º–µ—Ä (–∏–Ω–¥–µ–∫—Å), —Ç—ä—Ä—Å–∏—à –ø–æ –∏–º–µ (–∫–ª—é—á).

### –û—Å–Ω–æ–≤–Ω–∞ –∏–¥–µ—è:
–†–µ—á–Ω–∏–∫—ä—Ç —Å–≤—ä—Ä–∑–≤–∞ —É–Ω–∏–∫–∞–ª–Ω–∏ –∫–ª—é—á–æ–≤–µ —Å –∫–æ–Ω–∫—Ä–µ—Ç–Ω–∏ —Å—Ç–æ–π–Ω–æ—Å—Ç–∏:
–∫–ª—é—á ‚Üí —Å—Ç–æ–π–Ω–æ—Å—Ç

### –ü—Ä–∏–º–µ—Ä:
~~~~python
student = {
    "–∏–º–µ": "–ò–≤–∞–Ω",
    "–≤—ä–∑—Ä–∞—Å—Ç": 20,
    "–æ—Ü–µ–Ω–∫–∞": 5.75
}
~~~~

"–∏–º–µ", "–≤—ä–∑—Ä–∞—Å—Ç", "–æ—Ü–µ–Ω–∫–∞" —Å–∞ –∫–ª—é—á–æ–≤–µ
"–ò–≤–∞–Ω", 20, 5.75 —Å–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç–∏

### –û—Å–Ω–æ–≤–Ω–∏ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏:
| **–•–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞**| **–û–±—è—Å–Ω–µ–Ω–∏–µ**                                                  |
|-------------------|----------------------------------------------------------------|
| –¢–∏–ø	             | Dict                                                           |
| –†–µ–¥               | –û—Ç Python 3.7+, —Ä–µ—á–Ω–∏—Ü–∏—Ç–µ –∑–∞–ø–∞–∑–≤–∞—Ç —Ä–µ–¥–∞ –Ω–∞ –≤–º—ä–∫–≤–∞–Ω–µ            |
| –ö–ª—é—á–æ–≤–µ	         | –¢—Ä—è–±–≤–∞ –¥–∞ —Å–∞ —É–Ω–∏–∫–∞–ª–Ω–∏ –∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä str, int, tuple) |
| –°—Ç–æ–π–Ω–æ—Å—Ç–∏         | –ú–æ–≥–∞—Ç –¥–∞ —Å–∞ –≤—Å—è–∫–∞–∫—ä–≤ —Ç–∏–ø (–≤–∫–ª—é—á–∏—Ç–µ–ª–Ω–æ –¥—Ä—É–≥–∏ —Ä–µ—á–Ω–∏—Ü–∏)           |
| –î–æ—Å—Ç—ä–ø            | –ü–æ –∫–ª—é—á, –∞ –Ω–µ –ø–æ –∏–Ω–¥–µ–∫—Å (–∫–∞—Ç–æ –ø—Ä–∏ —Å–ø–∏—Å—ä—Ü–∏)                     |                                                        |


### –ü—Ä–∏–º–µ—Ä–Ω–∞ —É–ø–æ—Ç—Ä–µ–±–∞:
~~~~python
person = {
    "–∏–º–µ": "–ú–∞—Ä–∏—è",
    "–≤—ä–∑—Ä–∞—Å—Ç": 25,
    "–≥—Ä–∞–¥": "–ü–ª–æ–≤–¥–∏–≤"
}

person["–ø—Ä–æ—Ñ–µ—Å–∏—è"] = "—É—á–∏—Ç–µ–ª"  # –î–æ–±–∞–≤—è–Ω–µ –Ω–∞ –Ω–æ–≤ –∫–ª—é—á
~~~~

### –°—ä–∑–¥–∞–≤–∞–Ω–µ –Ω–∞ —Ä–∞—á–Ω–∏—Ü–∏
~~~~python
d = {}                          # –ø—Ä–∞–∑–µ–Ω
d = {"a": 1, "b": 2}            # –ª–∏—Ç–µ—Ä–∞–ª
d = dict(a=1, b=2)              # —á—Ä–µ–∑ –∫–ª—é—á–æ–≤–∏ –∞—Ä–≥—É–º–µ–Ω—Ç–∏
d = dict([("a", 1), ("b", 2)])  # –æ—Ç —Å–ø–∏—Å—ä–∫ –æ—Ç –¥–≤–æ–π–∫–∏
d = dict.fromkeys(["x", "y"], 0)# –µ–¥–Ω–∞–∫–≤–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç
~~~~

### –ö–ª—é—á–æ–≤–µ –∏ —Å—Ç–æ–π–Ω–æ—Å—Ç–∏:
–ö–ª—é—á–æ–≤–µ—Ç–µ —Ç—Ä—è–±–≤–∞ –¥–∞ —Å–∞ —Ö–µ—à–∏—Ä—É–µ–º–∏/–Ω–µ–∏–∑–º–µ–Ω—è–µ–º–∏ (–∏–º–º—É—Ç–∞–±–∏–ª–Ω–∏): 
**str, int, float, tuple** –æ—Ç —Ö–µ—à–∏—Ä—É–µ–º–∏ –∏ –¥—Ä.
–°—Ç–æ–π–Ω–æ—Å—Ç–∏—Ç–µ –º–æ–≥–∞—Ç –¥–∞ –±—ä–¥–∞—Ç –≤—Å—è–∫–∞–∫–≤–∏ –æ–±–µ–∫—Ç–∏.
**d = {("row", 1): ["a", "b"], 42: {"ok": True}}**

~~~~python
d = {("row", 1): ["a", "b"], 42: {"ok": True}}
~~~~
### –û–±—Ö–æ–∂–¥–∞–Ω–µ
~~~~python
for k in d: ...                 # –∫–ª—é—á–æ–≤–µ (–ø–æ –ø–æ–¥—Ä–∞–∑–±–∏—Ä–∞–Ω–µ)

~~~~
.keys()/.values()/.items() —Å–∞ –¥–∏–Ω–∞–º–∏—á–Ω–∏ –∏–∑–≥–ª–µ–¥–∏ ‚Äì –æ—Ç—Ä–∞–∑—è–≤–∞—Ç –ø—Ä–æ–º–µ–Ω–∏ –≤ —Ä–µ—á–Ω–∏–∫–∞.

### –°–æ—Ä—Ç–∏—Ä–∞–Ω–µ –Ω–∞ —Ä–µ—á–Ω–∏–∫

### –û–±—Ä—ä—â–∞–Ω–µ ‚Äû—Å—Ç–æ–π–Ω–æ—Å—Ç‚Üí–∫–ª—é—á‚Äú

## Dictionary comprehension:
Dictionary comprehension –µ –∫—Ä–∞—Ç—ä–∫ –∏ —É–¥–æ–±–µ–Ω –Ω–∞—á–∏–Ω –¥–∞ —Å—ä–∑–¥–∞–¥–µ—à —Ä–µ—á–Ω–∏–∫ –æ—Ç –∏—Ç–µ—Ä–∏—Ä—É–µ–º–∏ –¥–∞–Ω–Ω–∏ (—Å–ø–∏—Å—ä–∫, —Ä–µ—á–Ω–∏–∫, –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏ —Ç.–Ω.) 
–≤ –µ–¥–∏–Ω —Ä–µ–¥ ‚Äî —á—Ä–µ–∑ –∏–∑—Ä–∞–∑, –ø–æ–¥–æ–±–µ–Ω –Ω–∞ —Å–ø–∏—Å—ä–∫–æ–≤–∏—Ç–µ comprehension-–∏.
–°–∏–Ω—Ç–∞–∫—Å–∏—Å: **{–∫–ª—é—á: —Å—Ç–æ–π–Ω–æ—Å—Ç for –µ–ª–µ–º–µ–Ω—Ç in –∏—Ç–µ—Ä–∏—Ä—É–µ–º_–æ–±–µ–∫—Ç if —É—Å–ª–æ–≤–∏–µ}**
üîπ –∫–ª—é—á: ‚Äî –æ–ø—Ä–µ–¥–µ–ª—è –∫–∞–∫ —Å–µ –∏–∑—á–∏—Å–ª—è–≤–∞ –∫–ª—é—á—ä—Ç
üîπ —Å—Ç–æ–π–Ω–æ—Å—Ç ‚Äî –æ–ø—Ä–µ–¥–µ–ª—è –∫–∞–∫ —Å–µ –∏–∑—á–∏—Å–ª—è–≤–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç—Ç–∞
üîπ if ‚Äî (–ø–æ –∏–∑–±–æ—Ä) —Ñ–∏–ª—Ç—Ä–∏—Ä–∞ –µ–ª–µ–º–µ–Ω—Ç–∏

## 1.–û—Ç —Å–ø–∏—Å—ä–∫ ‚Üí —Ä–µ—á–Ω–∏–∫
~~~~python
nums = [1, 2, 3, 4, 5]
squares = {n: n**2 for n in nums}
print(squares)
# –ò–∑—Ö–æ–¥
{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
~~~~

## 2.–° —É—Å–ª–æ–≤–∏–µ(if)
~~~~python

print(even_squares)
# –ò–∑—Ö–æ–¥
{2: 4, 4: 16}
~~~~

## 3.–ü—Ä–µ–æ–±—Ä–∞–∑—É–≤–∞–Ω–µ –Ω–∞ –¥—Ä—É–≥ —Ä–µ—á–Ω–∏–∫
