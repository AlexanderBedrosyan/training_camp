`check_stock(stock, items)` –ø—Ä–æ–≤–µ—Ä—è–≤–∞ –¥–∞–ª–∏ `stock` –ø–æ–∫—Ä–∏–≤–∞ `items` (–∏ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞—Ç–∞). –í—Ä—ä—â–∞ —Å–ø–∏—Å—ä–∫ –Ω–∞ –Ω–µ–¥–æ—Å—Ç–∏–≥–∞—â–∏ –∞—Ä—Ç–∏–∫—É–ª–∏.

**–ü—Ä–∏–º–µ—Ä–µ–Ω —Ç–µ—Å—Ç:**
    ```python
    def check_stock(stock, items):
        shortage = []
        for item, need in items.items():
            if stock.get(item, 0) < need:
                shortage.append(item)
        return shortage

    print(check_stock({"–º–ª—è–∫–æ":10,"—Ö–ª—è–±":5}, {"–º–ª—è–∫–æ":8,"—Ö–ª—è–±":6}))
    # –û—á–∞–∫–≤–∞–Ω –∏–∑—Ö–æ–¥:
    ```

## –ó–∞–¥–∞—á–∞ 15: –ö–æ–º–±–∏–Ω–∏—Ä–∞–Ω–µ –Ω–∞ —Å–ø–∏—Å—ä—Ü–∏ –≤ —Ä–µ—á–Ω–∏–∫ –∏ —Å—Ä–µ–¥–Ω–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç

**–£—Å–ª–æ–≤–∏–µ:**  
`combine_to_dict(names, values)` –ø—Ä–∞–≤–∏ `{name: value}` –∏ –≤—Ä—ä—â–∞ —Ä–µ—á–Ω–∏–∫–∞ + —Å—Ä–µ–¥–Ω–∞—Ç–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç –Ω–∞ `values` (round 2).

**–ü—Ä–∏–º–µ—Ä–µ–Ω —Ç–µ—Å—Ç:**
    ```python
    def combine_to_dict(names, values):
        d = dict(zip(names, values))
        avg = round(sum(values)/len(values), 2) if values else 0
        return d, avg

    students = ["–ò–≤–∞–Ω","–ú–∞—Ä–∏—è","–ü–µ—Ç—ä—Ä"]
    scores = [85,92,78]
    print(combine_to_dict(students, scores))
    ```
